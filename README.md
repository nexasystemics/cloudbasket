# CloudBasket — Sovereign UI v1.0

## Unified Global Price Engine & Discovery Hub

CloudBasket is a premium price aggregator and affiliate discovery hub designed for the Indian and Global marketplace. Built with a focus on speed, privacy, and visual integrity.

### 🛡️ The Zero-Checkout Mandate
**CRITICAL:** CloudBasket is a "Zero-Checkout" platform. 
- No local shopping cart.
- No local checkout or payment processing.
- No storage of sensitive financial data (Stripe, etc.).
- All "Buy" actions are redirected through the **Income Shield** node to verified external affiliate partners (Amazon, Flipkart, etc.).

### 🚀 Tech Stack
- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript (Strict Mode)
- **Styling:** TailwindCSS 4 + Vanilla CSS
- **Database:** Supabase (PostgreSQL)
- **Compliance:** DPDPA 2023 (India), GDPR (EU), FTC (US)

### 🎨 Design Language
- **Aesthetic:** Apple-clean UI
- **Primary Colors:** Skyline Blue (#009DFF), Titanium Gray Accents
- **Dark Mode:** Zinc-950 Background (#09090b), Slate-200 Text (#e2e8f0)

### 📂 Key Routes
- `/products`: Global catalog with real-time filters.
- `/go/[id]`: Sovereign Redirect Node (Income Shield).
- `/admin`: Associate & Content management.
